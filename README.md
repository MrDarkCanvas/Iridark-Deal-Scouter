# Iridark Deal Scouter

Local-first deal intelligence platform focused on trusted coupons, discount codes, community insights, expiration tracking, and shopping intelligence.

## What is Iridark Deal Scouter?

Iridark Deal Scouter is a local-first deal intelligence application that helps users:

- Track coupons and discount codes with expiration dates and restrictions.
- Monitor deal quality and trustworthiness.
- Record fine print and requirements for each offer.
- Archive expired and historical offers to build long‑term shopping intelligence.
- Consult community findings and notes to avoid bad deals.
- Decide whether a deal is actually worth using or if there is a better alternative.

## Planned Features

### Coupon Intelligence
- Track trusted coupon sources.
- Manage coupon and discount code entries.
- Record fine print, restrictions, and expiration dates.
- Maintain a historical archive of coupons and discount codes.

### Community Intelligence
- Integrate community discussions and sentiment (e.g., Reddit).
- Collect community notes, complaints, and alternative recommendations.
- Observe reliability and patterns in community feedback.

### Scoring System
- Compute trust scores for coupon sources.
- Evaluate deal worthiness based on community insights and historical performance.
- Provide recommendations on whether to use a deal.

### Historical Archive
- Maintain a “coupon cemetery” of expired or failed offers.
- Track seasonal trends and sales patterns over time.

### Personal Intelligence
- Support user-defined favorite stores and categories.
- Keep a usage history and estimate money saved.
- Provide personalized deal recommendations.

## Technology Stack

- Python & Flask web framework
- SQLite database
- HTML, CSS & JavaScript for the web interface
