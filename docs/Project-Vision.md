# Project Vision

## Mission
Create a trustworthy shopping intelligence system that helps users make better purchasing decisions.

Not just finding coupons. Helping users determine:
- Is the deal real?
- Is it worth using?
- Is there a better option?
- Is there a catch?
- Should I buy now or wait?

## Core Principles
1. Trusted sources first.
2. Respect website rules and policies.
3. Preserve historical knowledge.
4. Community knowledge matters.
5. Local-first ownership of data.
6. Simplicity over unnecessary complexity.
7. Deal intelligence over coupon quantity.
8. Transparency in scoring.
9. Long-term usefulness over short-term hype.
10. User control over automation.

## Version 1 Goals
Version 1 should include:
- Local Flask application
- SQLite database
- Manual deal entry
- Manual coupon entry
- Trusted source tracking
- Expiration tracking
- Trust scores
- Deal worthiness scores
- Coupon cemetery
- Dashboard
- Search and filtering
- Favorite deals
- Basic reporting
